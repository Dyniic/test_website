The Product

Erode - AI is a web based platform built on top of the AWS stack. The platform uses a state of the art machine learning algorithm to deliver highly accuracte predictions on river bed erosion

The Stack

Erode-AI leverages AWS cloud services to deliver a solution that is secure and highly scalable.

The Data

The Data is extracted from the Canadian National Hydrographic Network

The Algorithm

The machine model was designed, tested, and trained at the University of Guelph as part of a doctoral study
