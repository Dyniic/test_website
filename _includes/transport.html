Transport

The roads on the estate are a bit rough and quite steep. We can traverse them in our little Fiat 500, so most of you should manage. We wouldn't recommend bringing a super low-riding car.

If you're an Uber user you'll be pleased to hear they operate in the Constantia/Hout Bay area. That said, please bear in mind that there are likely to be droves of guests leaving at roughly the same time and Constantia Nek is not a likely spot for a cabby to be hanging out. You might well run into a supply problem. Please team up with others if you can. If you've got a smart phone, but haven't tried out Uber yet, you can read more about it by clicking on the icon below. If you use the voucher code "nicolag610ue" when you download the app, you'll get some free credit.
